# AJRRADY Website

Official website of the **Association des Jeunes Ressortissants, Résidents et Amis pour le Développement de Youkounkoun (AJRRADY)**.

## Live Website

https://www.ajrrady.org

## About

AJRRADY is a non-profit organization dedicated to promoting the sustainable development of Youkounkoun through education, culture, health, sports, environmental initiatives, community solidarity, and diaspora engagement.

The website provides:

- Home page
- About AJRRADY
- Community Actions
- Projects & Achievements
- Events
- Photo Gallery
- Video Gallery
- Online Membership Application
- Contact Form
- SEO Optimization
- Responsive Design
- Google Search Console integration
- Google Business Profile integration
